Artikkeli: Command Line Basics Revisited

Alla on tiivistelmä Tero Karvisen artikkelista "Command Line Basics Revisited".

Linuxin komentotulkki on vanhempi kuin itse Linux ja Internet. Se on nopea ja tehokas työkalu tietokoneen hallintaan. Alla on lista tavallisista ja hyödyllisistä komentotulkin komennoista

Liikkuminen hakemistopuussa ja lukeminen

Komento Käyttö
pwdPrint Workin Directory - kertoo mikä on tämänhetkinen työkansio
ls listaa aktiivisen kansion tiedostot ja kansiot
cd [kansion nimi]/ muuttaa työkansion sen alikansioon
cd .. Näyttää tekstitiedoston tero.txt sisällön. Välilynti näyttää seuraavan sivun, b näyttää edellisen sivun, kenoviivalla "/" etsitään, "q" :lla lopetetaan.
less tero.txt
ls /etc|less less-pääte muuttaa komennon tulostuksen siten että nähdään yksi sivu kerralla

Tiedostomanipulointi

Komento Käyttö
nano F00.TXT avataan F00.TXT tiedostoa tekstieditorilla nano
mkdir NEWFOLDER luo kansion NEWFOLDER
mv OLDNAME NEWNAME siirtää tiedoston tai kansion tai muuttaa sen nimeä. Jos molemmat parametrit ovat tiedostoja, NEWNAME kirjoitetaan yli. Jos NEWNAME on kansio, OLDNAME siirty sinne.
cp -r ORIGINAL COPY kopioi ORIGINALin COPY kansioon. Merkit "-r" tarkoittaa rekursiivinen
rmdir EMPTYDIRpoistaa tyhjän kansion
rm JUNKpoistaa tiedoston JUNK
rm -r FOLDEROFJUNK poistaa kansion jossa voi olla sisältöä

SSH-etäyhteydet

Komento Käyttö
ssh tero@example.com avaa komentotulkin turvallisella etäyhteydellä. Käyttäjänimi on tero, serveri on example.com
remotecomputer$ exit kirjautuu ulos etäkoneesta
scp -r FOLDER tero@example.com:public_html/ kopioi turvallisesti kansion omasta koneesta etäkoneeseen

APU KOMENTOTULKIN KÄYTÖSSÄ

Komento Käyttö
man ls antaa manuaalin komennolle ls
ls --help apua komennon ls käytössä
wget -h apua komennon wget käytössä

HISTORIA JA ARVAAMINEN

Komento Käyttö
ls /etc/re[tab][tab]
reportbug.conf resolvconf/		painamalla tabulaattoria kaksi kertaa saadan komennon jatkomahdollisuudet
ls /etc/resolv.[tab] => ls /etc/resolv.conf tabulaattori kirjoittaa komennon loppuun jos sen jatko on yksiselitteinen
history näyttää komentohistorian

TÄRKEÄT TIEDOSTOT

Tiedosto Merkitys
/
reportbug.conf resolvconf/		Root, juurikansio, kansiopuun huippu. Kaikki kansiot ovat root kansion alikansioita
/home/ kaikkien käyttäjien kotikansiot ovat tässä
/home/erik/ Käyttäjän "erik" kotikansio
/etc/Järjestelmäasetukset ovat tässä kansiossa
/media/Kytkettävät mediat kuten /media/usbdisk
/var/log/ Järjestelmän logit ovat tässä kansiossa

ADMINKOMENNOT

Komento Käyttö
sudo apt-get updatepäivittää listan saatavilla olevista paketeista
apt-cache search [hakusanat] etsi ohjelmaa
sudo apt-get -y install [ohjelman nimi] asentaa ohjelman
dpkg --listfiles nethack-console => /usr/lib/games/nethack etsi komentokehotetta jolla käynnistetään nethack. Näyttää olevan "nethack"
sudo apt-get purge nethack poistaa ohjelman nethack
sudo apt-get -y install krita blender vlc tree httpie curl tmux python3-py asentaa monta ohjelmaa samanaikaisesti

Artikkeli: An Introduction To Data Science On The Linux Command Line

Alla on tiivistelmä Robert Elderin artikkelista "An Introduction To Data Science On The Linux Command Line".

Alla on otanta linuxin kommennoista joista voi olla hyötyä data-analyysissä.

Komento Käyttö
grep Voi löytää samanlaisia tekstipätkiä tiedostoista
sed Voi tehdä etsi ja korvaus - operaatioita
awk Voi tehdä edistyneitä etsi ja korvaus - operaatioita
sort Lajittelu
tsort Topologinen lajittelu
comm Joukko-operaatioita
uniq Voi tutkia yksikäsitteisyyttä
cat Liitosoperaatiot
head Tiedoston ensimmäisten rivien tulostaminen
tail Tiedoston viimeisten rivien tulostaminen
wc Sanojen laskeminen
find Etsintäoperaatiot
